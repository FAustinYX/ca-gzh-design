# 公司公众号视觉与编辑规范

## 样例结论

本规范来自 5 篇公司历史文章：USI 申请教程、签证体检信教程、UNSW O-Week 指南、Diploma CAL 风险分析、澳洲院校信息快讯。

历史内容已经形成三种稳定视觉信号：

- 蓝色：学校、活动、流程、信息与行动。
- 红色：风险、误区、门槛与紧迫性。
- 黄色：福利、提醒、注意事项和限时信息。

旧文章存在灰黄红模板，新文章存在蓝色与红色强视觉模板。公司定制版统一为“蓝色信息体系 + 红色风险体系”，黄色只作为功能性点睛，减少文章之间的品牌漂移。

## 色彩

| 用途 | 色值 | 规则 |
|---|---|---|
| 公司主蓝 | `#457EF2` | 教程、活动、快讯的章节与主锚点 |
| 深蓝 | `#1960CC` | 强行动、重要链接、深色标题 |
| 浅蓝 | `#D7E4FD` / `#F4F7FF` | 卡片背景、边框、关键词下划线 |
| 公司主红 | `#F03C16` | 风险洞察、误区、警示章节 |
| 深红 | `#B62727` | 高风险结论和重要门槛 |
| 提醒黄 | `#FBC12E` | 注意、福利、限时、材料准备 |
| 标题 | `#111827` | 主标题、章节标题 |
| 正文 | `#374151` | 正文统一使用 |
| 辅助文字 | `#6B7280` / `#9CA3AF` | 图注、标签、来源 |

禁止在同一篇文章中同时把蓝、红、黄都作为大面积主色。蓝或红只能选一个作为主色，黄色只点睛。

## 字体与留白

- 字体栈：系统中文无衬线字体。
- 正文：15px，行高 1.8，段后间距 18–22px。
- 章节标题：18–22px；封面标题不超过 28px。
- 手机端正文左右有效留白约 16–20px。
- 单段建议 40–120 个汉字；过长段落按语义拆分，但不改原意。
- 连续三个以上卡片后插入普通正文或留白，避免“卡片墙”。

## 组件使用

- 每篇只允许一个封面/引言强组件。
- 章节统一使用两位编号 `01/02/03`；结尾可用 `END` 或 `∞`。
- 操作教程的步骤必须连续编号，图片紧随对应步骤。
- 活动日程优先使用时间线或 Day 卡，不用长段落堆时间。
- 风险文章用“误区—事实—后果—解决方案”对照，不连续制造恐慌。
- 快讯按学校或机构分区，每条更新明确“变化、影响、行动”。
- CTA 仅在末尾出现一次；福利或群聊入口可在正文出现一次轻提示。

## 编辑语气

- 保留留学服务内容的亲近感，但事实段落保持清晰、可信。
- emoji 用作导航，不作装饰堆叠；同一段最多 1 个。
- 感叹号连续不超过 1 个；除封面钩子外不使用多重感叹。
- 英文学校名、课程名、签证术语首次出现时保留全称，后文可用常用简称。
- 政策与申请类文章不得把推测写成确定事实。
