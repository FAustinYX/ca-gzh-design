# 公司模板路由

## 判定顺序

依次判断文章是否以“风险论证”“操作步骤”“活动日程”“多条更新”为主。命中第一项即作为主类型；若内容复合，只在主主题内部选辅助组件。

## 1. 教程攻略

识别信号：申请、注册、准备材料、Step、操作截图、FAQ、资格条件。

默认主题：公司蓝。

样例：USI 申请、签证体检信申请。

组件配方：

1. 蓝色教程封面或简短引言。
2. “适合谁/为什么要办”说明。
3. 黄色准备清单。
4. 蓝色连续步骤卡；每步图片紧随正文。
5. 红色仅用于错误操作或硬性限制。
6. FAQ 卡。
7. 唯一咨询 CTA + 公司二维码。

长教程使用完整目录；短教程不生成目录，避免头重脚轻。

## 2. 活动指南

识别信号：O-Week、迎新、展会、时间地点、Day 1/2、路线、福利、报名或群聊。

默认主题：公司蓝。

样例：UNSW O-Week 迎新指南。

组件配方：

1. 情绪型蓝色封面。
2. 活动价值三点卡。
3. 时间、地点、适合人群信息条。
4. Day/时间线日程。
5. 黄色福利与领取提示。
6. 地图、地址或预约信息。
7. 唯一行动 CTA + 公司二维码；正文已有入群码时合并，不重复放码。

日程信息优先保证扫读效率，不把每个时间点做成重量相同的大卡片。

## 3. 风险洞察

识别信号：挂科、淘汰、风险、误区、门槛、为什么失败、反常识、课程推广。

默认主题：公司红。

样例：UNSW Diploma CAL 风险分析。

组件配方：

1. 反常识引言卡。
2. 风险形成机制。
3. 编号章节逐层论证。
4. 误区对照卡，最多 3 组连续出现。
5. 分数、费用、截止时间使用数据卡。
6. 解决方案与适用人群。
7. 唯一服务 CTA + 公司二维码。

红色用于结论和警示，不把所有关键词都标红；正文关键词使用浅红下划线。

## 4. 院校快讯

识别信号：本周更新、多所院校、多条政策变化、申请开放、语言要求、截止日期。

默认主题：公司蓝，使用紧凑配方。

样例：悉尼大学语言要求与 UNSW 申请开放快讯。

组件配方：

1. “本周更新”简洁封面。
2. 三条以内摘要导航。
3. 按学校分区，不按消息重要性混排。
4. 每条采用“变化 → 影响 → 行动”三段式。
5. 红色文字只标记不接受、截止、满位等风险词。
6. 表格与例外课程完整保留。
7. 唯一咨询 CTA + 公司二维码。

快讯通常篇幅短，不使用大面积目录、长引言或复杂签名卡。

## 复合内容

- 教程 + 风险：以教程为主，风险用公司蓝主题中的红色警告组件。
- 活动 + 福利：以活动为主，福利用黄色提示卡。
- 快讯 + 深度解读：更新条目超过两所学校时仍用快讯；单一政策展开论证时改为风险洞察或教程。
- 风险 + 产品推广：正文先完整论证，服务内容只在末尾 CTA 出现一次。
